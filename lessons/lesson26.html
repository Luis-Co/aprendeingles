Seasons & Weather

Level: A1
Length: 60 min
Topic: Weather, seasons, temperature
Score: 0%

Lesson Goals

By the end of this lesson, you will be able to:

  • Ask "What's the weather like?" & answer with correct adjectives & verbs
  • Use It is... / It's...ing accurately (sunny vs. it's raining)
  • Read & say temperatures (°C & °F) and extremes
  • Describe seasonal activities & clothes
  • Avoid classic errors: The weather is rain, it is snow

Input & Explanation

1. Core Question & Short Answer

What's the weather like today?
It's sunny / cloudy / windy / foggy / rainy / snowy / hot / cold / warm / cool / dry / humid.
It's raining / snowing / drizzling / pouring. (verbs = continuous)

2. Verbs vs Adjectives

Adjective (state) Verb (action now)
It's sunny. The sun is shining.
It's cloudy. It isn't raining now.
It's snowy. It is snowing outside.

NEVER double adjective + -ing: ✖️ It's sunnying → ✅ It's sunny / It's shining.

3. Temperature

20 °C = twenty degrees Celsius /ˈselsiəs/
68 °F = sixty-eight degrees Fahrenheit /ˈfærənhaɪt/
minus 5 °C = minus five (degrees)
zero degrees /ˈzɪərəʊ dɪˈɡriːz/
Extremes: boiling (≈35 °C+), freezing (≈0 °C), mild (≈15 °C)

4. Extra Phenomena & Intensifiers

a bit cold, quite windy, really humid, absolutely boiling
heavy rain / light snow / strong wind / thick fog
a shower / a storm / thunder and lightning / hail / drought / flood

5. Seasonal Vocabulary

Spring

blossom, light jacket, allergies

Summer

beach, sunglasses, sunscreen, picnic, fan, air-con

Autumn/Fall

leaves turn yellow, harvest, hoodie, cloudy

Winter

frost, ice, scarf, gloves, heater, ski, snowman

6. Typical Errors

❌ Wrong ✅ Correct
The weather is rain. It is rainy / It is raining.
It snows now. It is snowing now.
It's very boiling. It's absolutely boiling. (intensifier)
There is cold. It is cold.

2. Choose correct form:

  2. It (is sunny / is sunning) today.
  3. It (rains / is raining) every afternoon.
  4. There (is / is being) thick fog.
  5. It (is very / absolutely) freezing.
3. Read temperatures aloud:

  2. 72 °F →
