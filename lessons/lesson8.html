Lesson 8: There is / There are

Level: A1
Length: 60 min
Topic: Talking about existence, describing places
Score: 0%

Lesson Goals

By the end of this lesson, you will be able to:

  • Choose correctly between There is (singular) and There are (plural)
  • Form questions & negatives naturally
  • Use some / any with There is/are
  • Ask "How many... are there?" and answer with numbers & quantities
  • Describe a room, a street, or a picture without repeating nouns

Part A: Explanation & Examples

1. Basic form

Affirmative Negative Question Short answers
There is a cat. There isn't a cat. Is there a cat? Yes, there is. / No, there isn't.
There are two cats. There aren't two cats. Are there two cats? Yes, there are. / No, there aren't.

Key point: "There" is an introductory subject. The real subject comes after the verb.

2. Contractions in speech

There's a park. / There're /ðərər/ (fast) or There are (careful)

Informal spoken English: There's two apples and an orange. (spoken, not written exam)

3. Some / Any with There is/are

Affirmative Negative Question
There are some chairs. There aren't any chairs. Are there any chairs?

Special case: Use some in questions for offers and requests:

Would you like some coffee? (offer)

Can I have some water? (request)

4. How many... are there?

How many + plural noun + are there?

How many students are there in your class?
There are twenty. / There are a lot. / There are a few.

5. Prepositions of place (quick recap)

in, on, under, next to, between, behind, in front of, above, below

There is a lamp on the table.
There are some shoes under the bed.

6. Common errors

❌ Wrong ✅ Correct
There is three books. There are three books.
There isn't any chairs. There aren't any chairs.
Has a table in the room. There is a table in the room.
There have two cars. There are two cars.

Part B: Interactive Practice

1. Choose: is / are

a) There a big tree.
b) There two beds.
c) There some milk.

2. Make negative

There are two windows.

3. Add some or any

There isn't sugar left.
There are flowers in the garden.

4. Correct the mistakes

a) There is five chairs.
b) There have a lot of books.

Part E: Self-Study Roadmap

Grammar Focus

  • Practice using There is/There are correctly
  • Master the use of some/any with There is/are
  • Learn to form questions and negatives
  • Practice asking "How many... are there?"

Vocabulary

  • Learn furniture and room vocabulary
  • Study prepositions of place
  • Practice quantity words (a lot, a few, etc.)

Skills Practice

  • Reading: identify There is/are structures in texts
  • Listening: notice There is/are usage in conversations
  • Speaking: practice describing rooms and places
  • Writing: write descriptions using There is/are
